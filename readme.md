# Telegram Mini App для недвижимости

Готовый статический шаблон под GitHub Pages.

## Что внутри

- `index.html` — разметка одностраничника
- `styles.css` — стили
- `app.js` — логика формы, меню и webhook-запросов

## Разделы

- **Мои объявления**
  - форма создания объявления
  - список объявлений
  - удаление объявлений
- **Планировки**
  - отправка фото плана квартиры
  - prompt для генерации интерьера
- **Улучшить фото**
  - отправка фото объекта
  - prompt для улучшения
